The Arcade Tool allows your CrewAI agents to seamlessly integrate with 100+ third-party tools and services through Arcade's unified API platform. From Google Workspace and Slack to GitHub and social media platforms, Arcade provides authenticated access to a vast ecosystem of tools.

## **Installation**

```bash
pip install crewai-arcade
```

## **Setup**

### **Prerequisites**
1. **Arcade API Key**: Sign up at [arcade.dev](https://arcade.dev) to get your API key
2. **LLM API Key**: Required for agent functionality (OpenAI, Anthropic, Google, etc.)

### **Environment Variables**

You can set your API keys using environment variables or a `.env` file (recommended):

**Option 1: Using .env file (Recommended)**
```bash
# .env
ARCADE_API_KEY=your_arcade_api_key_here

# Choose your preferred LLM provider
OPENAI_API_KEY=your_openai_api_key_here
# or ANTHROPIC_API_KEY=your_anthropic_api_key_here
# or GEMINI_API_KEY=your_gemini_api_key_here
# or any other supported LLM provider
```

**Option 2: Using export commands**
```bash
export ARCADE_API_KEY="your_arcade_api_key_here"
export OPENAI_API_KEY="your_openai_api_key_here"  # or your preferred LLM provider
```

## **Basic Usage**

### **Quick Start Example**

Here's a simple example to get you started with Arcade tools in CrewAI:

```python
#!/usr/bin/env python
"""
Simple Arcade integration example for CrewAI
"""
import os
from crewai import Agent, Task, Crew, Process
from crewai_arcade import ArcadeToolManager
from crewai.llm import LLM

def main():
    # Step 1: Initialize Arcade Tool Manager
    print("🎮 Setting up Arcade tools...")
    manager = ArcadeToolManager(default_user_id="user@example.com")
    
    # Step 2: Get specific tools you need
    # This example uses Google Gmail and Slack tools
    tools = manager.get_tools(
        tools=["Google.ListEmails"],  # Specific Google tool
        toolkits=["Slack"]            # All Slack tools
    )
    print(f"📦 Loaded {len(tools)} tools from Arcade")

    # Step 3: Create your AI agent
    email_assistant = Agent(
        role="Email Assistant",
        goal="Help manage and organize email communications effectively",
        backstory="""You are a helpful email assistant who can read emails, 
        summarize important messages, and coordinate with team members via Slack.""",
        tools=tools,
        llm=LLM(model="gpt-4o-mini"),  # Use any supported LLM: gpt-4o, claude-3-5-sonnet, gemini-pro, etc.
        verbose=True
    )

    # Step 4: Define the task
    email_task = Task(
        description="""
        1. Check my latest emails from today
        2. Identify any urgent or important messages
        3. Summarize the key points and action items
        4. If there are urgent items, notify the team via Slack
        """,
        expected_output="""
        A clear summary containing:
        - Number of emails reviewed
        - List of important/urgent emails with sender and subject
        - Key action items that need attention
        - Confirmation if team was notified via Slack
        """,
        agent=email_assistant
    )

    # Step 5: Create and run the crew
    crew = Crew(
        agents=[email_assistant],
        tasks=[email_task],
        process=Process.sequential,
        verbose=True
    )

    print("\n🚀 Starting email management task...")
    result = crew.kickoff()
    
    print("\n✅ Task completed!")
    print("📋 Results:")
    print(result)

if __name__ == "__main__":
    main()
```

**To run this example:**

1. Create a new Python file (e.g., `arcade_example.py`)
2. Add your API keys to `.env` file
3. Install dependencies: `pip install crewai crewai-arcade`
4. Run: `python arcade_example.py`

⚠️ **First-time setup**: You'll need to authorize tools when prompted. The script will show authorization URLs in the console.

## **Custom Authentication Flow**

For production applications or when you need more control over the authorization process, implement a custom authentication flow:

```python
#!/usr/bin/env python
"""
Advanced Arcade integration with custom authentication flow
"""
import time
from typing import Any
from crewai import Agent, Task, Crew, Process
from crewai_arcade import ArcadeToolManager
from crewai.llm import LLM

class AdvancedArcadeManager:
    """Enhanced Arcade manager with custom authentication handling"""
    
    def __init__(self, user_id: str = "user@example.com"):
        self.user_id = user_id
        self.manager = ArcadeToolManager(executor=self._tool_executor)
        
    def _custom_auth_flow(self, manager: ArcadeToolManager, tool_name: str, **tool_input: dict[str, Any]) -> Any:
        """Enhanced authentication flow with better UX"""
        print(f"\n🔐 Checking authorization for {tool_name}...")
        
        # Check current authorization status
        auth_response = manager.authorize(tool_name, self.user_id)

        if not manager.is_authorized(auth_response.id):
            print(f"⚠️  Tool '{tool_name}' requires authorization")
            print(f"🌐 Please visit: {auth_response.url}")
            print("📱 Complete the authorization in your browser")
            print("⏳ Waiting for authorization (this may take a moment)...")
            
            # Wait for authorization with timeout
            start_time = time.time()
            timeout = 300  # 5 minutes
            
            while time.time() - start_time < timeout:
                auth_response = manager.wait_for_auth(auth_response)
                if manager.is_authorized(auth_response.id):
                    print(f"✅ Successfully authorized {tool_name}")
                    return
                time.sleep(2)
            
            raise TimeoutError(f"❌ Authorization timeout for {tool_name}")
        else:
            print(f"✅ {tool_name} already authorized")

    def _tool_executor(self, tool_manager: ArcadeToolManager, tool_name: str, **tool_input: dict[str, Any]) -> Any:
        """Execute tool with custom auth flow"""
        self._custom_auth_flow(tool_manager, tool_name, **tool_input)
        return tool_manager.execute_tool(self.user_id, tool_name, **tool_input)
    
    def get_tools(self, tools: list[str] = None, toolkits: list[str] = None):
        """Get tools with error handling"""
        try:
            return self.manager.get_tools(tools=tools, toolkits=toolkits)
        except Exception as e:
            print(f"❌ Error loading tools: {e}")
            return []

def main():
    # Initialize advanced manager
    arcade_manager = AdvancedArcadeManager(user_id="user@example.com")
    
    # Load tools for comprehensive workflow automation
    tools = arcade_manager.get_tools(
        tools=["Google.ListEmails", "Google.SendEmail", "Google.CreateEvent"],
        toolkits=["Slack", "GitHub"]
    )
    
    if not tools:
        print("❌ No tools loaded. Please check your Arcade API key.")
        return

    # Create executive assistant agent
    executive_assistant = Agent(
        role="Executive Assistant",
        goal="Streamline communications and automate workflows across platforms",
        backstory="""You are an experienced executive assistant with access to 
        email, calendar, messaging, and development tools. You excel at managing 
        communications, scheduling, and coordinating with development teams.""",
        tools=tools,
        llm=LLM(model="gpt-4o"),  # Use any supported LLM provider
        verbose=True
    )

    # Define comprehensive workflow task
    workflow_task = Task(
        description="""
        Daily workflow automation:
        1. Review today's emails and categorize by importance
        2. Check for any meeting requests and schedule appropriately  
        3. Look for any urgent GitHub notifications or mentions
        4. Send a brief status update to the team via Slack
        5. Create calendar events for any follow-ups needed
        """,
        expected_output="""
        A comprehensive daily report including:
        - Email summary with categorization (urgent/normal/low priority)
        - Calendar events created or updated
        - GitHub activity summary
        - Slack communication sent
        - List of follow-up actions scheduled
        """,
        agent=executive_assistant
    )

    # Execute the workflow
    crew = Crew(
        agents=[executive_assistant],
        tasks=[workflow_task],
        process=Process.sequential,
        verbose=True
    )

    print("\n🚀 Starting daily workflow automation...")
    try:
        result = crew.kickoff()
        print("\n✅ Workflow completed successfully!")
        print("📊 Results:")
        print(result)
    except Exception as e:
        print(f"\n❌ Workflow failed: {e}")

if __name__ == "__main__":
    main()
```

**Key improvements in this advanced example:**

- **Better error handling**: Timeout management and graceful failures
- **Enhanced UX**: Clear progress indicators and status messages  
- **Production-ready**: Proper class structure and modular design
- **Comprehensive workflow**: Real-world executive assistant scenario

### **Authorization Management**
```python
# Check authorization status before execution
def check_tool_auth(manager: ArcadeToolManager, tool_names: list[str], user_id: str):
    """Check which tools need authorization"""
    needs_auth = []
    
    for tool_name in tool_names:
        auth_response = manager.authorize(tool_name, user_id)
        if not manager.is_authorized(auth_response.id):
            needs_auth.append((tool_name, auth_response.url))
    
    return needs_auth

# Use in your workflow
unauth_tools = check_tool_auth(manager, ["Google.ListEmails", "Slack.SendMessage"], USER_ID)
if unauth_tools:
    print("Please authorize these tools:")
    for tool_name, auth_url in unauth_tools:
        print(f"- {tool_name}: {auth_url}")
```

### **Error Handling**
```python
def robust_tool_execution(manager: ArcadeToolManager, tool_name: str, user_id: str, **kwargs):
    """Execute tool with comprehensive error handling"""
    try:
        # Check authorization first
        auth_response = manager.authorize(tool_name, user_id)
        if not manager.is_authorized(auth_response.id):
            return {"error": f"Tool {tool_name} requires authorization: {auth_response.url}"}
        
        # Execute tool
        result = manager.execute_tool(user_id, tool_name, **kwargs)
        return {"success": True, "data": result}
        
    except Exception as e:
        return {"error": f"Tool execution failed: {str(e)}"}
```

## **Troubleshooting**

### **Common Issues**

**Authorization Failures**
- Ensure you visit the authorization URL provided in the console
- Check that your Arcade API key is valid and has the necessary permissions
- Some tools may require additional setup in their respective platforms

**Tool Conflicts**
- Avoid loading similar tools that might conflict (e.g., multiple email tools)
- Be specific about which tools you need rather than loading entire toolkits

**Rate Limiting**
- Some integrated services have rate limits
- Implement appropriate delays and error handling
- Consider batching operations when possible

**Environment Variables**
```bash
# Verify your environment variables are set
echo $ARCADE_API_KEY
echo $OPENAI_API_KEY # or $ANTHROPIC_API_KEY, $GEMINI_API_KEY, etc.

# Or check your .env file
cat .env | grep -E "(ARCADE_API_KEY|OPENAI_API_KEY)"
```
